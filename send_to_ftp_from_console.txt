отправить файл на ФТП из консоли

curl -u moyLogin:parol-Ot-FTP -T monitoring.txt ftp://88.88.88.88/72CD/ten/

-u - для указания логина и пароля
-T - для отправки файла
ftp://88.88.88.88  -фтп ресурс
/72CD/ten/         -путь внутри ФТП - кладет туда

Важно - не забывать поставить "/" после всего, если не закрыть слешом - ничего пересылаться не будет
например, если будет так - ftp://88.88.88.88/72CD/ten   - то файл не перешлется
