Настраиваем SSH



устанавливаем его в систему, он может быть уже установлен, а дальше правим конфиг, предварительно сделав копию 
nano sshd_config
раскомментируем там строчки

PasswordAuthentication yes
Port 22
Protocol 2

сохраняем и закрываем, стартуем сервис (предполагается, что перед правкой конфига он был остановлен)
service sshd start

все, дальше пробуем подключиться
ssh ВашИпАдресс -p 22 -l ИмяПользователяВсистеме

в UBUNTU 20.04 может возникнуть проблема, если нет подключения:
apt install openssh-server — устанавливаем пакет
systemctl status ssh — проверяем, запущен ли
ufw allow ssh — отключаем фаервол
iptables -I INPUT -p tcp –dport 22 -m state –state NEW -j ACCEPT – разрешаем 22 порт
